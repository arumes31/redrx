How We Use Data

This page describes how information is processed by Redrx to provide its services.

Link Analytics

When you click a shortened link, we collect minimal data to provide analytics to the link creator. This includes:

  • IP Address: Used to determine approximate geographic location (Country level only) and to count unique clicks.
  • User Agent: Used to identify the browser and operating system.
  • Referrer: Identifies which website the click originated from.

Security & Stability

We process technical data to ensure the platform remains secure and available for everyone:

  • Rate Limiting: IP addresses are used to prevent abuse and automated spam by limiting the number of requests per minute.
  • Phishing Protection: Destination URLs are checked against global blocklists to prevent the distribution of malicious links.

Third-Party Processing

To run our service efficiently, we use these specialized providers:

  • Cloudflare: Provides security filtering and DDoS protection.
  • MaxMind: Provides the database used to convert IP addresses into country names locally on our servers.

Data Persistence

Analytics data is stored in our database to show you click trends. Session information for logged-in users is stored in a temporary cookie that expires when you log out.
